Question-1: What are the different ways to select an element in the DOM?

there are many ways to select an element in DOM. but the main ways are: document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName(), document.querySelector() and document.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: it gets or sets the HTML content inside an element, including all HTML tags.
innerText: it gets or sets the visible text content of an element, excluding hidden elements and respecting CSS styling.
textContent: it gets or sets the text content of an element, including all child nodes, regardless of visibility or CSS styling

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where you delegate the handling of events to a parent element rather than attaching event listeners to individual child elements

Question-4: What is event bubbling in the DOM?

Event bubbling is a key concept in the DOM (Document Object Model) that describes how events propagate through the elements. When an event occurs on a specific element (a button click), it doesn't just trigger on that element alone. Instead, the event "bubbles up" through its ancestor elements in the DOM tree, triggering any event listeners attached to those ancestors.?
